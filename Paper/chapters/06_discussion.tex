\chapter{深入讨论}\label{chap:discussion}

本节针对两个最具启发性的现象展开讨论：时序模态下监督与无监督方法的性能反转，以及主动防御的失效边界，并在此基础上给出部署建议。

\section{时序模态下的性能反转}

Exp2 显示，时序模态在重度污染下出现了显著的性能反转：监督组的平均 AUC-ROC 跌幅高达约 19.7 个百分点，而无监督组整体不降反升（约 $+1.16$ 个百分点，见~\ref{tab:exp2-drop}）。这意味着随着噪声加剧，原本在干净标注下落后的无监督方法将反超监督方法。该现象可从以下三方面理解：

\begin{enumerate}
    \item \textbf{无监督方法对标签噪声的结构性免疫}。重构型与距离型无监督方法（如自编码器、KNN、Matrix Profile）在训练时\emph{完全不读取标签}，其优化目标仅为最小化重构误差或刻画子序列距离：
    \begin{equation}
        \mathcal{L}_{\text{Recon}}=\|X_{\text{ts}}-\hat{X}_{\text{ts}}\|_F^2 .
    \end{equation}
    因此，无论标签如何翻转，其学到的表征不受影响——这正是其在高污染下保持稳定的根本原因。
    \item \textbf{监督循环模型对翻转标签的过拟合}。时序异常通常为稀有事件（异常占比常低于 5\%）。在 20\% 对称污染下，翻转产生的伪异常数量甚至可能超过真实异常，使交叉熵损失的梯度方向被伪正/伪负样本主导，模型易退化为平凡解，时间上下文表征随之崩坏。
    \item \textbf{监督方法在时序上的优势本就脆弱}。即便在干净标注下（Exp1），循环自编码器 LSTM-AE（0.738）已优于有监督的 LSTM-Sup（0.687），而表现最佳的是无需标签训练的卷积特征方法 MiniRocket（0.809）。这表明在时序模态，监督信号带来的增益有限，一旦标签受污染便迅速被噪声抵消。
\end{enumerate}

\section{主动防御的失效边界}

Exp4 表明，\texttt{IQR\_Trim} 对 MLP 与树集成模型带来正向挽救，却对逻辑回归（$-1.21$ 个百分点）与 TabPFN（$-0.75$ 个百分点）产生负迁移；若进一步在所有清洁器上平均，二者的损失更为明显。这一分化恰是“没有免费的午餐”原则的具体体现，其机制如下：

\begin{enumerate}
    \item \textbf{TabPFN：上下文密度被破坏}。TabPFN 是参数固定的预训练 Transformer，推理时将整个训练集作为上下文，通过注意力执行近似的贝叶斯后验推断：
    \begin{equation}
        P(y_\ast\mid x_{\text{test}},\mathcal{D})\approx\mathrm{Softmax}\big(\mathrm{Attn}(Q,K,V)\big).
    \end{equation}
    其预测质量依赖训练集的局部密度结构。样本剪裁人为抹除了部分样本，破坏了上下文的密度分布，干扰了注意力的后验积分，反而削弱了其本就较强的抗噪能力。
    \item \textbf{逻辑回归：边界支撑点流失}。LR 的分离超平面高度依赖紧贴决策边界的关键样本。\texttt{IQR\_Trim} 基于单维分位数边界（式~\ref{eq:iqr_range}）剔除离群点，容易误删那些处于两类交界、对边界定位至关重要的“少数派”样本，导致线性决策面发生漂移与旋转，从而产生负迁移。
\end{enumerate}

由此可见，主动剪裁防御的收益取决于下游模型的归纳偏置：对易过拟合噪声的高容量非线性模型有益，而对依赖完整样本密度或边界支撑点的模型则可能有害。

\section{工业部署建议}

基于上述分析，我们对高噪声、多模态场景下的异常检测部署给出以下建议：
\begin{enumerate}
    \item \textbf{按模型容量匹配去噪策略}。对 MLP、GBDT 等高容量、易过拟合的下游模型，建议启用 \texttt{IQR\_Trim} 或 \texttt{IForest\_Trim} 前置防御，剪裁比例可参照估计的污染率。
    \item \textbf{对线性与上下文学习模型谨慎去噪}。若下游为逻辑回归、线性 SVM 或 TabPFN，应避免激进剪裁，转而采用 L2 正则、鲁棒损失或扩大上下文宽度等被动抗噪手段。
    \item \textbf{在时序等高难度模态优先考虑无监督方法}。当标注质量难以保证时，标签免疫的无监督/重构方法在鲁棒性上更具优势。
    \item \textbf{保留小规模“黄金验证集”}。建议人工核验保留占比 1\%--5\% 的纯净验证集，用于动态标定剪裁比例与判定阈值，避免在含噪验证集上调参导致的过拟合。
\end{enumerate}
